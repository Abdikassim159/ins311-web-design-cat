QUESTION 2: Front-end Website Design (15 marks)
====================================================================

PROJECT: Agricultural Tips Website

TECHNOLOGIES USED:
- HTML: Page structure
- CSS: Styling and responsive layout
- JavaScript: Form validation and CRUD operations
- XML: Data storage (data.xml)

====================================================================
HOW IT WORKS
====================================================================

1. HTML creates 4 pages: Home, Services, Agri-Tips, Contact

2. CSS makes the website responsive (works on phone, tablet, laptop)

3. JavaScript handles:
   - Creating new tips
   - Reading tips from XML
   - Updating existing tips
   - Deleting tips
   - Contact form validation

4. XML (data.xml) stores agricultural tips that load when page opens

====================================================================
FILES INCLUDED
====================================================================

- index.html (Home page)
- services.html (Services page)
- agritips.html (Agri-Tips with CRUD)
- contact.html (Contact form)
- style.css (All styling)
- script.js (JavaScript functions)
- data.xml (XML data file)

====================================================================
HOW TO RUN
====================================================================

1. Open index.html in any web browser
2. Click navigation links to view all pages
3. On Agri-Tips page: Add, Edit, or Delete tips
4. On Contact page: Fill form and submit

====================================================================
XML USAGE
====================================================================

- File: data.xml
- Contains agricultural tips in XML format
- JavaScript reads this file using DOMParser
- Tips display on Agri-Tips page
